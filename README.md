# Zdoc

Another markup language for note.

## 词汇定义

* `Zdoc` 表示该标记语言。
* `ZdocTool` 表示用于翻译和处理该标记语言的工具软件。

## 目的

用于为 ZhiZe 提供知识管理系统的文件格式基础。

* 表达能力
  * 可用于给自然语言纯文本标记增加格式和样式说明。
  * 可以结构化表达数据，用于内嵌图表和配置信息等。
  * 可以内嵌不经修改或经过简单修改的其他编程语言代码块。
* 语法复杂度
  * 比 XML 更简单。
  * 比 Markdown 更统一和方便扩展。
* 运行环境
  * 渲染后具有交互性和信息更新能力。
  * 代码自动整理。
  * 多文件结构和文件系统访问能力。
  * 内嵌代码块的设定环境运行能力。
